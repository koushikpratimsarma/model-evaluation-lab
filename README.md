This repository is my learning journey to understand how NLP models and Large Language Models have evolved over time.

Instead of only learning the latest AI models, I wanted to understand **why each model was created**, **what problem it solved**, and **why researchers moved to the next model**.

The notebooks in this repository follow the historical evolution of NLP, starting from simple rule-based systems and ending with modern AI Agents.

### Learning Roadmap

* Rule-Based Systems
* Bag of Words (BoW)
* TF-IDF
* Word2Vec
* GloVe
* FastText
* Sequence Problem
* Recurrent Neural Network (RNN)
* Long Short-Term Memory (LSTM)
* Gated Recurrent Unit (GRU)
* Sequence-to-Sequence (Seq2Seq)
* Attention Mechanism
* Transformer
* BERT
* GPT
* Modern Large Language Models
* Retrieval-Augmented Generation (RAG)
* AI Agents

### Why I Built This

I created this repository to strengthen my understanding of NLP and modern AI.

My goal is not just to know how to use models, but also to understand how they evolved and why each new architecture was introduced.

I hope this repository will also help others who are starting their AI learning journey.
